    vim/vi操作
    1.跳到文本的最后一行：按“G”,即“shift+g”
    2.跳到最后一行的最后一个字符 ： 先重复1的操作即按“G”，之后按“$”键，即“shift+4”。
    3.跳到第一行的第一个字符：先按两次“g”，
    4.跳转到当前行的第一个字符：在当前行按“0”。
    显示行号：
    :set number

    向文件内追加内容：

    cat >> file.txt <<EOF
    helloworld
    EOF

    解析：
    1.>> 是标准输出追加重定向符号

    1. << 是标准输入重定向符号
    2. EOF 标志符，这可以任意指定，但开始和结尾都要是同一个标识符
