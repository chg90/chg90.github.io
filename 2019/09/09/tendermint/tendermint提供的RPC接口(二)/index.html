    /tx_search

    说明：获取未确认的交易列表

    参数：

    query，查询条件，必填

    prove，是否包含块中交易的证明，非必填，默认false

    page，第几页，非必填，默认为1

    per_page，每页多少数据，非必填，默认30

    这个接口和/unconfirmed_txs相比，是可以分页的

    示例：

     C:\Users\ch>curl -s "localhost:26657/tx_search?query=\"tx.height=10\"&prove=true&page=1&per_page=10"
{
  "jsonrpc": "2.0",
  "id": "",
  "result": {
    "txs": [],
    "total_count": "0"
  }
}
